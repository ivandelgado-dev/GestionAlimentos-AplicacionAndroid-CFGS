# Gestión de Alimentos
Descripción

Gestión de Alimentos es una aplicación Android que permite gestionar ComponenteDieta e Ingredientes. La app ofrece funcionalidades de CRUD, navegación entre pantallas, y persistencia de datos mediante Room, utilizando Jetpack Compose para la interfaz y una arquitectura por capas.

Funcionalidades
Básicas
 - Crear, leer, actualizar y eliminar ComponenteDieta.
 - Crear, leer, actualizar y eliminar Ingredientes.
 - Navegación entre diferentes pantallas de la app.
 - Persistencia de datos local con Room.

Avanzadas
 - Gestión de listas de objetos de manera dinámica.
 - UI moderna con Jetpack Compose.
 - Uso de ViewModel y Flow para manejar datos reactivos.
 - Arquitectura organizada en capas (UI, dominio, datos).

Requisitos técnicos
 - Lenguaje: Kotlin (principal) y Java (interoperabilidad).
 - UI: Jetpack Compose.
 - Persistencia: Room y Flow.
 - Arquitectura: Capas separadas para UI, dominio y datos.
 - Gestión de datos: Objetos de ComponenteDieta e Ingredientes con listas dinámicas.

Notas
 - Se requiere un dispositivo Android con compatibilidad para Jetpack Compose.
 - Room gestiona la persistencia local, no se necesita conexión a internet.
 - La app permite explorar múltiples casos de uso de ComponenteDieta e Ingredientes.
